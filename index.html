역함수 관계와 대칭
📐
기울기 $-1$ 직선과 $\sqrt{2}$ 법칙
지수함수와 로그함수가 역함수 관계일 때, 기울기 -1인 직선 위의 두 교점은 반드시 y=x에 대해 대칭입니다.

두 점의 $x$좌표 차이 $= d$

→ 두 점 사이의 거리 $= d\sqrt{2}$

일반 공식: 기울기 $m$인 직선 위 두 점 거리 $= \Delta x \cdot \sqrt{1+m^2}$

$m = -1$ 대입 → $\Delta x \cdot \sqrt{2}$

기울기 -1인 직선이 지수·로그함수 쌍과 만날 때
교점 좌표를 직접 구하지 않아도 거리 계산 즉시 가능
🔄
등차↔등비 변환 구조
그래프 위 점들의 좌표를 설정할 때 x/y 중 어느 것이 등차·등비를 이루는지 즉시 판단합니다.

$y = a^x$ → $x$가 등차수열 $\Rightarrow$ $y$는 등비수열

$y = \log_a x$ → $x$가 등비수열 $\Rightarrow$ $y$는 등차수열

일정 간격 점들: x, x+d, x+2d 또는 x, xr, xr² 로 고정 후 빠르게 연립 세팅
📏
로그함수 '넓이 보존' 법칙
y=logₐx 와 y=logₐ(kx) 는 평행이동 관계. 두 그래프와 두 수평선으로 둘러싸인 넓이는 적분 없이 구합니다.

넓이 $= (c_2 - c_1) \times \log_a k$

이유: 두 그래프의 가로 간격이 어느 높이에서도 일정 $(= \log_a k)$

적분 계산 없이 평행사변형 넓이 공식처럼 처리
대소 비교 · 부등식
🔍
대소 비교 경계값 대입 강령
두 곡선의 교점 x₁의 범위를 추론할 때, 특정 경계값 x=k를 두 식에 각각 대입하여 y값의 위아래(대소 관계)가 어떻게 바뀌는지 확인합니다.

예: "$x_1 > \frac{1}{2}$ 인가?" 검증 방법

→ $x = \frac{1}{2}$ 를 두 함수에 각각 대입

→ $f\!\left(\frac{1}{2}\right)$ vs $g\!\left(\frac{1}{2}\right)$ 대소 관계 확인

→ 교점 전후로 대소가 뒤집히는지 체크

ㄱ/ㄴ/ㄷ 진위 판정형, 두 곡선 교점의 위치 추론 문제
대략 찍는 게 아니라 수학적으로 정석인 행동 강령. 경계값 대입 하나로 해결
⚠️
로그 부등식 — 밑이 변수일 때 케이스 분류
밑에 변수가 들어가면 밑의 범위에 따라 부등호 방향이 뒤집힙니다. 이걸 즉석에서 생각하면 반드시 실수 납니다.

$\log_{f(x)} g(x) > 0$ 꼴:

① $f(x) > 1$ 일 때: $g(x) > 1$ (부등호 유지)

② $0 < f(x) < 1$ 일 때: $0 < g(x) < 1$ (부등호 유지, 범위 다름)

+ 공통 조건: $f(x) > 0,\; f(x) \neq 1,\; g(x) > 0$ 반드시 체크

케이스 분류 표를 머릿속에 정형화해두지 않으면 시험장에서 반드시 실수

쌍곡선함수 합성 구조
$f(x) = \dfrac{a^x+a^{-x}}{2}$, $g(x) = \dfrac{a^x-a^{-x}}{2}$ 꼴이 조건으로 나올 때 이 성질을 모르면 연립에 5분 낭비.

$f(x)$ → $y$축 대칭 (우함수)

$g(x)$ → 원점 대칭 (기함수)

[핵심 항등식] $\{f(x)\}^2 - \{g(x)\}^2 = 1$ (항상 성립)

증명

$\{f(x)\}^2 - \{g(x)\}^2$

$= \left[\dfrac{a^x+a^{-x}}{2}\right]^2 - \left[\dfrac{a^x-a^{-x}}{2}\right]^2$

$= \dfrac{2a^x}{2} \cdot \dfrac{2a^{-x}}{2} = a^x \cdot a^{-x} = 1$ ✓

🔲
격자점 문제 — 부등식 영역화와 층별 분할
지수·로그 곡선 사이의 격자점을 하나씩 세다가는 시험 종이 칩니다.

치트키 1: $a^x \leq n$ 꼴 → 무조건 $\log$ 변환 후 정수 구간 처리

$a^x \leq n \;\Rightarrow\; x \leq \log_a n$

치트키 2: $y = a^x$ 와 $y = \log_a x$ (역함수 관계) 사이의

격자점 개수 $= y=x$ 아래쪽 격자점 $\times 2 +$ 대칭축 위의 점

y축 방향으로 층별로 잘라서 각 층의 x 범위를 정수 구간으로 처리

사인/코사인 4등분 박스 법칙
한 주기를 직사각형 박스로 감싸면 4개의 합동 영역으로 쪼개집니다. 넓이 계산·대칭성 추론의 기반.

한 주기 = 4개의 합동 직사각형 박스

각 박스의 가로 $= \dfrac{T}{4}$ ($T$: 주기)

$1:\sqrt{3}$ 비율: 수평선 $y=k$가 박스와 만날 때

→ 대칭축으로부터 교점까지의 거리 비율이 특수각과 연결

🎯
대칭축을 이용한 교점의 합 공식
y=sin(ωx)와 수평선 y=k의 교점 x₁, x₂를 직접 구하지 않아도 합을 알 수 있습니다.

$x_1 + x_2 = 2 \times$ (가장 가까운 대칭축의 $x$좌표)

예: $y=\sin(x)$와 $y=0.5$의 교점 $x_1,\,x_2$ $(0 < x < \pi)$

→ 대칭축 $x = \dfrac{\pi}{2}$

→ $x_1 + x_2 = \pi$

원리

$\sin(\alpha) = \sin(\pi-\alpha)$ 이므로

두 교점은 항상 삼각함수 대칭축에 대해 대칭.

$x_1 + x_2 = 2 \times$ (대칭축) 자동 성립.

방정식의 근의 합을 그래프 대칭축으로 눈으로 읽음 → 계산 없이 연립 해소
🔀
합성 공식 역방향 실전
기본서는 "합성 후 단순화"를 가르치지만, 수능에서는 역방향이 훨씬 강력합니다.

$a\sin\theta + b\cos\theta$ 의 최댓값 $= \sqrt{a^2 + b^2}$

역방향 적용:

"최댓값이 $k$이다" 조건 → $a^2 + b^2 = k^2$

→ 계수 관계를 연립방정식으로 바로 세팅

최댓값/최솟값이 조건으로 주어지고, a, b 중 일부를 역산해야 할 때
합성 과정 자체를 생략. "최댓값$^2 = a^2 + b^2$"만 기억하면 됨
🔵
단위원 대칭 이동 독해법
sin/cos 각 변환을 공식 암기 대신 단위원 위의 점의 이동으로 파악합니다. 해의 개수 문제를 2초 안에 시각화.

$\sin(\pi - x) = \sin x$ → $y$축 대칭

$\sin(\pi + x) = -\sin x$ → 원점 대칭

$\sin(-x) = -\sin x$ → $x$축 대칭

$\cos\!\left(\dfrac{\pi}{2} + x\right) = -\sin x$ → 90° 반시계 회전

삼각방정식 해의 개수 → 단위원 위에 수평선/수직선 하나 긋는 것으로 2초 컷
사인법칙 · 코사인법칙
⭕
외접원과 지름의 직각삼각형 활용
문제에 외접원이 나오고 원주각이 보이면 십중팔구 사인법칙이 출발점입니다.

사인법칙: $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$

외접원 지름 $= 2R$ 이면:

→ 지름의 끝점에서 보이는 원주각 $= 90°$

→ 직각삼각형 생성 → 피타고라스/삼각비 활용

외접원 반지름 R이 주어지거나 구해야 할 때
$2R = \dfrac{a}{\sin A}$ → $R = \dfrac{a}{2\sin A}$. 외접원 나오면 이 식부터 쓸 것
🔷
내접 사각형 — 마주보는 각의 합 180°
원에 내접하는 사각형의 마주보는 두 각의 합은 항상 π. 이를 이용해 대각선을 공유하는 두 삼각형에 코사인 법칙을 연립합니다.

원에 내접하는 사각형 ABCD:

$\angle A + \angle C = \pi,\quad \angle B + \angle D = \pi$

$\angle C = \pi - \angle A$ 이므로

$\cos(\angle C) = \cos(\pi - \angle A) = -\cos(\angle A)$

→ 대각선 AC를 공유하는 △ABC와 △ACD에

각각 코사인 법칙 적용 후 연립 ($\cos\angle A = -\cos\angle C$ 이용)

원에 내접하는 사각형 + 대각선 길이 또는 넓이를 구하는 문제
⊙
할선 정리 (교차하는 현)
삼각함수 활용 단원이지만 원 안에서 두 현이 교차할 때 성립하는 중학 기하 성질이 4점짜리 결정적 단서로 등장합니다.

두 현 AC, BD가 원 안의 점 P에서 교차할 때:

$PA \cdot PC = PB \cdot PD$

원 밖의 점 P에서 할선 두 개를 그을 때:

$PA \cdot PB = PC \cdot PD$ (A, B는 한 할선의 교점, C, D는 다른 할선)

원과 직선이 나오고 길이 관계를 묻는 문제 → 이 공식이 핵심 단서일 가능성 높음
📐
사영 정리 (제1코사인법칙)
교과서에서 사라진 공식이지만 수능 도형 문제에서 한 줄 선형 연립을 만들어줍니다.

삼각형 ABC에서 A에서 BC에 수선의 발을 내릴 때:

$$a = b\cos C + c\cos B$$

"한 변 = 다른 두 변을 바닥에 사영한 길이의 합"

유도

코사인 법칙 $b^2 = a^2+c^2-2ac\cos B$ 에서 $c$에 대해 정리,

$c^2 = a^2+b^2-2ab\cos C$ 에서 $b$에 대해 정리하여 더하면

$a = b\cos C + c\cos B$ 가 나옵니다 (교육과정 내 수준).

제곱(a²) 없이 선형 방정식. 두 변이 주어지고 세 번째 변을 구할 때 연립을 단순화
△
삼각형 넓이 3종 세트 + 코사인 이차방정식

넓이 기본: $S = \dfrac{1}{2}ab\sin C$

넓이 외접원: $S = \dfrac{abc}{4R}$

넓이 내접원: $S = \dfrac{(a+b+c) \cdot r}{2}$

코사인 법칙의 $c$에 대한 이차방정식:

$c^2 - (2b\cos A)c + (b^2-a^2) = 0$

→ 두 근의 합 $= 2b\cos A$, 두 근의 곱 $= b^2-a^2$

세 변과 외접원 $R$이 주어질 때 $\dfrac{abc}{4R}$로 넓이 한 줄 처리. $c$를 모를 땐 이차방정식 세워서 근과 계수 관계 사용

📈
$S_n$ — 상수항 없는 이차함수 관점

$$S_n = \frac{d}{2}n^2 + \left(a_1 - \frac{d}{2}\right)n \quad \leftarrow \text{상수항} = 0 \text{ 항상!}$$

꼭짓점(대칭축): $n = \dfrac{d - 2a_1}{2d}$ 근처 자연수

원점 외 교점 $(S_n=0)$: 꼭짓점 $n$의 정확히 2배

"$S_{11}$에서 최댓값" → $S_{22}$ 또는 $S_{23}$ 근처에서 $S_n=0$으로 꺾임. 대입 없이 구조 파악 가능

$a_{n+1} - a_n = f(n)$ 꼴을 1부터 $n-1$까지 대입해서 더하는 노가다를 $\Sigma$ 한 줄로 압축합니다.

$a_{n+1} - a_n = f(n)$ 꼴일 때:

$$a_n = a_1 + \sum_{k=1}^{n-1} f(k)$$

(단, $n \geq 2$. $n=1$은 별도 확인)

유도 (축차대입)

$a_2 - a_1 = f(1)$

$a_3 - a_2 = f(2)$

$\quad\vdots$

$a_n - a_{n-1} = f(n-1)$

모두 더하면: $a_n - a_1 = \displaystyle\sum_{k=1}^{n-1} f(k)$

따라서 $a_n = a_1 + \displaystyle\sum_{k=1}^{n-1} f(k)$

이 공식을 외워두면 1~(n-1) 대입 노가다 없이 바로 Σ 계산으로 점프

역함수 정적분 — ㄱ자 넓이

$$\int_a^b f(x)\,dx + \int_{f(a)}^{f(b)} f^{-1}(x)\,dx = b\cdot f(b) - a\cdot f(a)$$

기하학적 의미

$y=f(x)$와 $y=f^{-1}(x)$로 둘러싸인 넓이가

큰 직사각형 $(b \times f(b))$에서 작은 직사각형 $(a \times f(a))$을 뺀

"ㄱ자 모양 넓이"와 같음.

원함수 적분 + 역함수 적분이 동시에 나오는 문제 → 이 공식으로 연산 없이 처리

등가속도 운동 — 평균속도 공식

$v(t)$가 1차식(등가속도)일 때:

$t_1 \sim t_2$ 구간 이동 거리

$= \dfrac{v(t_1) + v(t_2)}{2} \times (t_2 - t_1)$

$=$ 사다리꼴 넓이!
