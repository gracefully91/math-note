역함수 관계와 대칭
📐
기울기 -1 직선과 √2 법칙
지수함수와 로그함수가 역함수 관계일 때, 기울기 -1인 직선 위의 두 교점은 반드시 y=x에 대해 대칭입니다.
두 점의 x좌표 차이 = d → 두 점 사이의 거리 = d√2 일반 공식: 기울기 m인 직선 위 두 점 거리 = Δx · √(1+m²) m = -1 대입 → Δx · √2
기울기 -1인 직선이 지수·로그함수 쌍과 만날 때
교점 좌표를 직접 구하지 않아도 거리 계산 즉시 가능
🔄
등차↔등비 변환 구조
그래프 위 점들의 좌표를 설정할 때 x/y 중 어느 것이 등차·등비를 이루는지 즉시 판단합니다.
y = aˣ → x가 등차수열 ⟹ y는 등비수열 y = logₐx → x가 등비수열 ⟹ y는 등차수열
일정 간격 점들: x, x+d, x+2d 또는 x, xr, xr² 로 고정 후 빠르게 연립 세팅
📏
로그함수 '넓이 보존' 법칙
y=logₐx 와 y=logₐ(kx) 는 평행이동 관계. 두 그래프와 두 수평선으로 둘러싸인 넓이는 적분 없이 구합니다.
넓이 = (c₂ - c₁) × logₐk 이유: 두 그래프의 가로 간격이 어느 높이에서도 일정(= logₐk)
적분 계산 없이 평행사변형 넓이 공식처럼 처리
대소 비교 · 부등식
🔍
대소 비교 경계값 대입 강령
두 곡선의 교점 x₁의 범위를 추론할 때, 특정 경계값 x=k를 두 식에 각각 대입하여 y값의 위아래(대소 관계)가 어떻게 바뀌는지 확인합니다.
예: "x₁ > 1/2 인가?" 검증 방법 → x = 1/2 를 두 함수에 각각 대입 → f(1/2) vs g(1/2) 대소 관계 확인 → 교점 전후로 대소가 뒤집히는지 체크
ㄱ/ㄴ/ㄷ 진위 판정형, 두 곡선 교점의 위치 추론 문제
대략 찍는 게 아니라 수학적으로 정석인 행동 강령. 경계값 대입 하나로 해결
⚠️
로그 부등식 — 밑이 변수일 때 케이스 분류
밑에 변수가 들어가면 밑의 범위에 따라 부등호 방향이 뒤집힙니다. 이걸 즉석에서 생각하면 반드시 실수 납니다.
log_{f(x)} g(x) > 0 꼴: ① f(x) > 1 일 때: g(x) > 1 (부등호 유지) ② 0 < f(x) < 1 일 때: 0 < g(x) < 1 (부등호 유지, 범위 다름) + 공통 조건: f(x) > 0, f(x) ≠ 1, g(x) > 0 반드시 체크
케이스 분류 표를 머릿속에 정형화해두지 않으면 시험장에서 반드시 실수
심화 · 격자점
∞
쌍곡선함수 합성 구조
f(x) = (aˣ+a⁻ˣ)/2, g(x) = (aˣ-a⁻ˣ)/2 꼴이 조건으로 나올 때 이 성질을 모르면 연립에 5분 낭비.
f(x) → y축 대칭 (우함수) g(x) → 원점 대칭 (기함수) [핵심 항등식] {f(x)}² - {g(x)}² = 1 (항상 성립)
증명
{f(x)}² - {g(x)}² = [(aˣ+a⁻ˣ)/2]² - [(aˣ-a⁻ˣ)/2]² = [aˣ+a⁻ˣ + aˣ-a⁻ˣ]/2 · [aˣ+a⁻ˣ - aˣ+a⁻ˣ]/2 = (2aˣ/2) · (2a⁻ˣ/2) = aˣ · a⁻ˣ = 1 ✓
🔲
격자점 문제 — 부등식 영역화와 층별 분할
지수·로그 곡선 사이의 격자점을 하나씩 세다가는 시험 종이 칩니다.
치트키 1: aˣ ≤ n 꼴 → 무조건 log 변환 후 정수 구간 처리 aˣ ≤ n → x ≤ logₐn 치트키 2: y = aˣ 와 y = logₐx (역함수 관계) 사이의 격자점 개수 = y=x 아래쪽 격자점 × 2 + 대칭축 위의 점
y축 방향으로 층별로 잘라서 각 층의 x 범위를 정수 구간으로 처리

사인/코사인 4등분 박스 법칙
한 주기를 직사각형 박스로 감싸면 4개의 합동 영역으로 쪼개집니다. 넓이 계산·대칭성 추론의 기반.
한 주기 = 4개의 합동 직사각형 박스 각 박스의 가로 = T/4 (T: 주기) 1:√3 비율: 수평선 y=k가 박스와 만날 때 → 대칭축으로부터 교점까지의 거리 비율이 특수각과 연결
🎯
대칭축을 이용한 교점의 합 공식
y=sin(ωx)와 수평선 y=k의 교점 x₁, x₂를 직접 구하지 않아도 합을 알 수 있습니다.
x₁ + x₂ = 2 × (가장 가까운 대칭축의 x좌표) 예: y=sin(x)와 y=0.5의 교점 x₁, x₂ (0 < x < π) → 대칭축 x = π/2 → x₁ + x₂ = π
원리
sin(α) = sin(π-α) 이므로 두 교점은 항상 삼각함수 대칭축에 대해 대칭. x₁ + x₂ = 2 × (대칭축) 자동 성립.
방정식의 근의 합을 그래프 대칭축으로 눈으로 읽음 → 계산 없이 연립 해소
🔀
합성 공식 역방향 실전
기본서는 "합성 후 단순화"를 가르치지만, 수능에서는 역방향이 훨씬 강력합니다.
a·sinθ + b·cosθ 의 최댓값 = √(a² + b²) 역방향 적용: "최댓값이 k이다" 조건 → a² + b² = k² → 계수 관계를 연립방정식으로 바로 세팅
최댓값/최솟값이 조건으로 주어지고, a, b 중 일부를 역산해야 할 때
합성 과정 자체를 생략. "최댓값² = a² + b²"만 기억하면 됨
🔵
단위원 대칭 이동 독해법
sin/cos 각 변환을 공식 암기 대신 단위원 위의 점의 이동으로 파악합니다. 해의 개수 문제를 2초 안에 시각화.
sin(π - x) = sin x → y축 대칭 sin(π + x) = -sin x → 원점 대칭 sin(-x) = -sin x → x축 대칭 cos(π/2 + x) = -sin x → 90° 반시계 회전
삼각방정식 해의 개수 → 단위원 위에 수평선/수직선 하나 긋는 것으로 2초 컷

⭕
외접원과 지름의 직각삼각형 활용
문제에 외접원이 나오고 원주각이 보이면 십중팔구 사인법칙이 출발점입니다.
사인법칙: a/sinA = b/sinB = c/sinC = 2R 외접원 지름 = 2R 이면: → 지름의 끝점에서 보이는 원주각 = 90° → 직각삼각형 생성 → 피타고라스/삼각비 활용
외접원 반지름 R이 주어지거나 구해야 할 때
2R = a/sinA → R = a/(2sinA). 외접원 나오면 이 식부터 쓸 것
🔷
내접 사각형 — 마주보는 각의 합 180°
원에 내접하는 사각형의 마주보는 두 각의 합은 항상 π. 이를 이용해 대각선을 공유하는 두 삼각형에 코사인 법칙을 연립합니다.
원에 내접하는 사각형 ABCD: ∠A + ∠C = π, ∠B + ∠D = π ∠C = π - ∠A 이므로 cos(∠C) = cos(π - ∠A) = -cos(∠A) → 대각선 AC를 공유하는 △ABC와 △ACD에 각각 코사인 법칙 적용 후 연립 (cos∠A = -cos∠C 이용)
원에 내접하는 사각형 + 대각선 길이 또는 넓이를 구하는 문제
⊙
할선 정리 (교차하는 현)
삼각함수 활용 단원이지만 원 안에서 두 현이 교차할 때 성립하는 중학 기하 성질이 4점짜리 결정적 단서로 등장합니다.
두 현 AC, BD가 원 안의 점 P에서 교차할 때: PA · PC = PB · PD 원 밖의 점 P에서 할선 두 개를 그을 때: PA · PB = PC · PD (A, B는 한 할선의 교점, C, D는 다른 할선)
원과 직선이 나오고 길이 관계를 묻는 문제 → 이 공식이 핵심 단서일 가능성 높음
📐
사영 정리 (제1코사인법칙)
교과서에서 사라진 공식이지만 수능 도형 문제에서 한 줄 선형 연립을 만들어줍니다.
삼각형 ABC에서 A에서 BC에 수선의 발을 내릴 때: a = b·cosC + c·cosB "한 변 = 다른 두 변을 바닥에 사영한 길이의 합"
유도
코사인 법칙 b² = a²+c²-2ac·cosB 에서 c에 대해 정리, c² = a²+b²-2ab·cosC 에서 b에 대해 정리하여 더하면 a = b·cosC + c·cosB 가 나옵니다 (교육과정 내 수준).
제곱(a²) 없이 선형 방정식. 두 변이 주어지고 세 번째 변을 구할 때 연립을 단순화
△
삼각형 넓이 3종 세트 + 코사인 이차방정식
넓이 기본: S = (1/2)ab·sinC 넓이 외접원: S = abc / (4R) 넓이 내접원: S = (a+b+c)·r / 2 코사인 법칙의 c에 대한 이차방정식: c² - (2b·cosA)c + (b²-a²) = 0 → 두 근의 합 = 2b·cosA, 두 근의 곱 = b²-a²
세 변과 외접원 R이 주어질 때 abc/(4R)로 넓이 한 줄 처리. c를 모를 땐 이차방정식 세워서 근과 계수 관계 사용

📈
S_n — 상수항 없는 이차함수 관점
S_n = (d/2)n² + (a₁ - d/2)n ← 상수항 = 0 항상! 꼭짓점(대칭축): n = -(a₁-d/2)/(d) = (d/2-a₁)/d 근처 자연수 원점 외 교점 (S_n=0): 꼭짓점 n의 정확히 2배
"S₁₁에서 최댓값" → S₂₂ 또는 S₂₃ 근처에서 S_n=0으로 꺾임. 대입 없이 구조 파악 가능
|S|
S_n 절댓값 조건 — 기하학적 대칭성
|S_n| = |S_m| 또는 S_k = S_{2k} 같은 조건은 식을 세우는 게 아니라 이차함수 그래프의 대칭성으로 해결합니다.
S_n = An² + Bn (상수항 0, 대칭축: n = -B/(2A)) |S_n| = |S_m| 조건: → 그래프 위 두 점의 "절댓값 높이"가 같음 → 대칭축에 대해 대칭이거나 x축에 대해 대칭 → 가능한 경우: n+m = 2×(대칭축) 또는 n, m이 S_n=0 교점 중 하나
식을 전개해서 n, m을 연립하면 시간 폭발. 반드시 그래프로 먼저 파악할 것

개차수열 일반화 공식 (점화식 축차대입)
a_{n+1} - a_n = f(n) 꼴을 1부터 n-1까지 대입해서 더하는 노가다를 Σ 한 줄로 압축합니다.
a_{n+1} - a_n = f(n) 꼴일 때: a_n = a_1 + Σ[k=1 to n-1] f(k) (단, n ≥ 2. n=1은 별도 확인)
유도 (축차대입)
a_2 - a_1 = f(1) a_3 - a_2 = f(2) ⋮ a_n - a_{n-1} = f(n-1) ───────────────────── 모두 더하면: a_n - a_1 = Σ[k=1 to n-1] f(k) 따라서 a_n = a_1 + Σ[k=1 to n-1] f(k)
이 공식을 외워두면 1~(n-1) 대입 노가다 없이 바로 Σ 계산으로 점프

a_{n+1}=f(a_n) 킬러 수열에서 역추적 수형도 그리기 전에 반드시 이 작업 먼저.
고정점: f(x) = x 를 만족하는 x → 수열이 도달하면 영원히 머무는 값 주기 궤도: f(f(x)) = x 이지만 f(x) ≠ x 인 x → 두 값을 왔다갔다 하는 2주기 수열 가능
실전 순서: 1단계: 고정점 찾기 → 가능한 a₁의 범위 좁히기 2단계: 주기 궤도 존재 여부 확인 3단계: 역추적 수형도 그려서 a₁의 개수 세기

로피탈 정리 (L'Hôpital's Rule)
교육과정 외지만 다항함수 0/0 꼴에서 미분계수 형태로 묶기 어려울 때 3초 컷.
lim[x→a] f(x)/g(x) (0/0 또는 ∞/∞ 꼴) = lim[x→a] f'(x)/g'(x) (이 극한이 존재하면)
수능에서 채점상 불이익은 없으나, 미분한 후의 극한이 존재하는 조건을 반드시 확인해야 안전. 다항함수 범위에서만 사용.

인수 개수 법칙 — 연속·미분가능
불연속 함수에 (x-α)ⁿ을 곱할 때 n의 개수가 결과를 결정합니다.
(x-α)¹ 곱하기 → 끊긴 점을 연속으로 (x-α)² 곱하기 → 꺾인 첨점을 미분가능하게 (접함) (x-α)³ 곱하기 → 변곡점 형태로 통과 역방향: 그래프가 α에서 어떻게 보이느냐로 → 필요한 인수 개수를 역산
조건부 함수의 미분가능성 문제에서 인수를 몇 개 곱해야 하는지 즉시 판단 가능
🔍
합성함수 h(x)=f(g(x)) 미분불가능점 2단계
1단계: g(x) 자체가 미분불가능한 x=a 전부 체크 → h'(a) 존재 여부 별도 확인 필요 2단계: g(c) = b 이고 f가 x=b에서 미분불가능한 모든 x=c 를 찾아서 체크
⚠️ 핵심 예외: 2단계에서 g'(c) = 0 이면 f'(b)가 없어도 h'(c) = f'(g(c))·g'(c) = 0 으로 미분가능할 수 있음. 이 케이스 빠뜨리면 오답!

삼차함수 극댓값·극솟값 차이 공식
도함수의 근을 알 때 극댓값과 극솟값의 차이를 식에 대입하지 않고 구합니다.
f(x) = ax³ + ··· 의 도함수 f'(x) = 0 의 두 근이 α, β (α < β) 일 때: 극댓값 - 극솟값 = |a| · (β-α)³ / 2 (도함수 이차함수의 넓이 공식에서 유도)

접점 → 교점 공식 (−2t − p)
x=t에서의 접선이 삼차함수와 다시 만나는 점을 미분·연립 없이 1초에 찾습니다. 대치동 킬러 치트키.
f(x) = x³ + px² + qx + r 에서 x = t에서의 접선이 만나는 나머지 교점의 x좌표: s = −2t − p
증명
f(x) - [x=t에서의 접선] = (x-t)² · (x-s) 좌변 전개 시 x²의 계수 비교: p - (-2t) = -(2t+s) 즉 2t + s = -p → s = -2t - p ✓
접점 x=t 와 최고차항 이차계수 p 만 알면 나머지 교점 즉시 확정. 연립방정식 한 줄 절약

6분의 공식 — 이차함수와 직선
∫[α→β] a(x-α)(x-β) dx = −a/6 · l³ 넓이 S = |a|/6 · l³ (l = β − α) 적용: 두 교점의 x좌표 차이 l만 알면 ∫ 기호 없이 한 줄 계산
이차함수와 x축, 또는 이차함수와 직선 사이의 넓이 문제에서 무조건 이 공식 먼저
⑫
12분의 공식 — 삼차함수와 접선
∫[α→β] a(x-α)²(x-β) dx = −a/12 · l⁴ 넓이 S = |a|/12 · l⁴ (l = β − α) 1:2 비율 관계와 결합 시 시너지 극대화 → 접점·교점만 알면 넓이 즉시 계산
㉚
30분의 공식 시리즈 — 사차함수
W자 바닥형: a(x-α)²(x-β)² → 넓이 = |a|/30 · l⁵ 3중근 접선형: a(x-α)³(x-β) → 넓이 = |a|/20 · l⁵ 한 접점 두 교점: a(x-α)²(x-β)(x-γ) (특수형) → 20분의 공식 변형 (접점이 정중앙일 때)
분모 정리표: 이차 × 1차 (6분의): l³ / 6 삼차 (12분의): l⁴ / 12 사차 3중근 (20분의): l⁵ / 20 사차 W바닥 (30분의): l⁵ / 30
✂️
평행이동으로 인수 단순화 — 핵심 행동 강령
지저분한 숫자가 나오면 x축 방향 평행이동으로 인수를 x 꼴로 먼저 바꾸기: ∫[2→5] (x-2)²(x-5) dx → x축 방향 -2 이동 ∫[0→3] x²(x-3) dx → 12분의 공식: |a|/12 · 3⁴ = 81/12 = 27/4
근거
정적분은 평행이동에 불변: ∫[a→b] f(x)dx = ∫[a+c→b+c] f(x-c)dx
항상 구간이 0에서 시작하도록 이동시킬 것. 계산 실수 원천 차단

삼차함수 세 근 사이 넓이 (변형 공식)
f(x) = a(x-α)(x-β)(x-γ) (세 근 α < β < γ) ∫[α→β] f(x) dx = a/12 · (β-α)³ · (4β-3α-γ) 특수 케이스 — 등간격 (β-α = γ-β = d): ∫[α→β] f(x) dx = a/4 · d⁴
세 근이 등간격이면 4분의 공식으로 한 번에 처리

변곡접선 통짜 넓이 공식
삼차함수 f(x)에서 변곡점을 지나는 직선이 곡선과 만드는 두 영역의 통짜 넓이: 두 교점 사이 거리 = l 일 때 통짜 넓이 S = |a|/4 · l⁴ (양쪽 각각 |a|/8 · l⁴)
삼각함수 대칭축으로 자른 면적 = 대칭이므로 각 부분은 |a|/8 · l⁴. 인테그랄 완전 소멸

심프슨 공식 (이차·삼차함수에서 정확 성립)
∫[a→b] f(x) dx = (b-a)/6 · [f(a) + 4f(m) + f(b)] (m = (a+b)/2 : 구간 중점) ⚠️ 이차·삼차함수에서만 정확히 성립 사차 이상은 근삿값
왜 이차·삼차에서 정확한가?
삼차 이하의 다항함수는 심프슨 규칙의 오차항이 정확히 0이 됩니다. → 6분의 공식의 일반화 형태
식을 연립하기 어렵지만 세 점 (a, f(a)), (m, f(m)), (b, f(b))의 높이만 기하적으로 알 때 한 줄 컷
↩️
역함수 정적분 — ㄱ자 넓이
∫[a→b] f(x)dx + ∫[f(a)→f(b)] f⁻¹(x)dx = b·f(b) − a·f(a)
기하학적 의미
y=f(x)와 y=f⁻¹(x)로 둘러싸인 넓이가 큰 직사각형(b×f(b))에서 작은 직사각형(a×f(a))을 뺀 "ㄱ자 모양 넓이"와 같음.
원함수 적분 + 역함수 적분이 동시에 나오는 문제 → 이 공식으로 연산 없이 처리
🔁
주기함수 정적분 평행이동
f(x+p) = f(x) (주기 p)이면: ∫[a→a+p] f(x)dx = ∫[0→p] f(x)dx (구간 무관, 항상 동일) ∫[2024→2026] f(x)dx (주기 2인 경우) → ∫[0→2] f(x)dx 로 강제 이동
아무리 더러운 숫자가 나와도 주기를 빼서 가장 계산 편한 대칭 구간으로 이동
↔️
우함수·기함수 적분 변환
∫[-a→a] 기함수 dx = 0 ∫[-a→a] 우함수 dx = 2·∫[0→a] 우함수 dx 우함수 적분 → 기함수 기함수 적분 → 우함수 (상수항 때문에 y축 대칭이나 원점 통과 아닐 수 있음)
대칭 조건 있는 함수를 정적분 함수 F(x)로 확장할 때 그래프 개형을 미분 없이 파악
🚗
속도·위치·거리 — 삼각형 넓이로 처리
v(t)가 직선(1차식)이면 이동 거리를 적분 대신 삼각형·사다리꼴 넓이 공식으로 처리: 이동 거리 = v(t) 그래프와 t축 사이의 넓이 (방향 전환 시 각 구간별로 넓이 합산) v(t) = 0이 되는 t를 먼저 찾아 구간 분리
위치 변화량 = ∫v dt (부호 포함), 이동 거리 = ∫|v| dt (절댓값). 둘을 혼동하지 말 것
⏱️
등가속도 운동 — 평균속도 공식
v(t)가 1차식(등가속도)일 때: t₁~t₂ 구간 이동 거리 = 평균 속도 × 걸린 시간 = [v(t₁) + v(t₂)]/2 × (t₂ - t₁) = 사다리꼴 넓이!
적분 연산 없이 초·말 속도의 평균 × 시간으로 즉시 계산

이중접선 1:1 법칙 (W자 사차함수)
W자 사차함수에서 두 극소점을 동시에 지나는 "공통 접선(이중접선)"이 존재할 때: 두 접점 α, β 사이의 중점 = 가운데 극대점의 x좌표 → 극대점 = (α + β) / 2 (1:1 내분)
좌우 비대칭 W자에서도 성립. 이중접선이 나오면 두 접점의 중점이 극대점
